Getting started

Welcome to the Tenderwell knowledge base. This space will hold product documentation, tutorials, howtos, and other helpful resources for Tenderwell.

Datasets​

Tenderwell provides 2 main datasets which may be useful to you: tenders and organizations. Both data sets are constantly being expanded with new tender sources being added weekly. You can check our statistics on the home page or by doing some searches in our search engines.

Tenders​

We currently have over 30mln tenders in our data set with some 30K added daily. Our main focus is in open and forecast tenders as these are the most relevant for our clients.

Organizations​

The organizations data set is mix of buyers, contracting authorities, and suppliers. This data set is built based on information found in public tender award notices and other tender notices.

Usage and Limitations​

Our platform is designed to be used by Tenderwell tender bidders and suppliers. As such we have set reasonable usage limits which both allow our users to find the information they need and ensure the good performance of the platform. If you need increased usage rates or custom subsets of data get in touch with us.
